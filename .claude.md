## Progetto LaraGoci

LaraGoci è un AI Companion Device basato su ESP32-S3-BOX-3 con firmware XiaoZhi.
Il device si connette via WebSocket al gateway OpenClaw e usa l'agente come cervello.

### Obiettivo attuale

Sviluppare l'estensione `extensions/xiaozhi/` dentro il repo OpenClaw (~/openclaw).
Questa estensione fa da bridge tra il firmware XiaoZhi (ESP32) e l'agent loop OpenClaw.

### Architettura in breve

ESP32 (firmware XiaoZhi) --WSS--> OpenClaw Gateway (bridge xiaozhi) --> Agente Claude

- Il firmware parla protocollo XiaoZhi (7 msg JSON + Opus binary)
- Il bridge traduce: audio Opus → Whisper STT → agente → TTS → Opus
- Il device usa la sessione `main` (stessa di WhatsApp/browser)
- Modello architetturale: estensione `extensions/voice-call/` (Twilio)
